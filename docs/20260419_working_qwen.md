### 4. tmux 通知の最適化 ✅

**発見**: `tmux send-keys` はエージェントの入力ストリームを汚す

**解決策**: `tmux display-message` を使用

```python
# 送信通知
tmux display-message "📤 AgentPost → gemini: メッセージ"
```

**利点**:
- ✅ ステータスバーに表示（入力を邪魔しない）
- ✅ 一時的（数秒で自動消去）
- ✅ 安全（編集中のコードに影響しない）

## 📝 技術的な学び

### 1. tmux send-keys の正しい使い方

```bash
# 推奨：別々のコマンド
tmux send-keys -t 1:3 "メッセージ" && tmux send-keys -t 1:3 C-m

# または
tmux send-keys -t 1:3 "メッセージ" C-m
```

### 2. テストの独立性

グローバル状態（BASE_DIR など）をテストで書き換えるのは困難。

**教訓**: 依存性注入パターンを検討すべき。

### 3. エラーハンドリング

キーの存在チェックを忘れると KeyError:
```python
# ❌ ダメ
ts = datetime.fromisoformat(msg["timestamp"]...)

# ✅ 良い
if "timestamp" not in msg:
    continue
ts = datetime.fromisoformat(msg["timestamp"]...)
```
