## 🎯 あなたの担当テストケース

### TC-002: MCP サーバー機能テスト

**テスト内容**:
- MCP サーバーを起動
- MCP サーバー経由でディスカッションを作成
- MCP サーバー経由でディスカッション一覧を取得
- MCP サーバー経由でディスカッション詳細を取得
- MCP サーバー経由でディスカッションに返信
- MCP サーバー経由でディスカッションを検索

**期待結果**:
- 全ての機能が正常に動作
- エラーメッセージが表示されない
- 返信が正常に投稿される

---

### TC-004: エラーメッセージ表示テスト

**テスト内容**:
- 成功時にエラーメッセージが表示されない
- エラー時は詳細が表示される

**テスト手順**:
1. 正常な返信を投稿：`github-discuss reply <URL> "返信"`
2. 無効な URL で返信を試みる：`github-discuss reply "invalid-url" "返信"`

**期待結果**:
- 成功時：「✅ コメントを追加しました！」のみ表示（エラー詳細は表示されない）
- 失敗時：エラー詳細が表示される
